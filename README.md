## Deployment Instructions

### GitHub Setup

1. Create new repository on GitHub:
   ```
   https://github.com/new
   Repository name: legal-sicario-website
   ```

2. Push files:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Legal Sicario website"
   git remote add origin https://github.com/YOUR_USERNAME/legal-sicario-website.git
   git push -u origin main
   ```

### Netlify Setup

1. Go to [https://app.netlify.com](https://app.netlify.com)
2. Click "New site from Git"
3. Connect GitHub account
4. Select `legal-sicario-website` repository
5. Build settings:
   - Build command: (leave empty — static site)
   - Publish directory: . (root)
6. Click "Deploy site"
7. Site goes live automatically!

## Updates

To update the website:
